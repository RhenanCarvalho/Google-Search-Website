Projeto Acadêmico: SEO e Google Search Console

Grupo de alunos do curso de Sistemas de Informação

Objetivo do Site

Este site foi criado como parte de um trabalho acadêmico para demonstrar na prática o uso do Google Search Console (GSC).

O que será feito?

  • Submeter o site ao Google Search Console
  • Monitorar cliques, impressões e indexação
  • Corrigir possíveis problemas de SEO

Sobre o Projeto

Com este estudo de caso, buscamos entender como a ferramenta GSC pode ser aplicada mesmo em sites simples como este. Acompanhe os resultados e relatórios que surgirão após a indexação.
